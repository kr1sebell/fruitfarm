# Фруктовая ферма

Прототип браузерной игры на React без сборки. Код написан так, чтобы его можно было использовать в WebView мобильных приложений, а также встроить в мини-приложения VK и Telegram.

## Запуск

Откройте `index.html` в браузере. Все зависимости подгружаются с CDN.

## Возможности

* Регистрация по логину и паролю (просто ввод имени).
* Начальный баланс 100 монет, первый участок и дерево.
* Деревья дают фрукты каждую секунду, если их поливали за последние 30 секунд.
* Фрукты можно собирать, складировать в ящики и продавать.
* Деревья и участки можно покупать и улучшать за монеты.

Код рассчитан на демонстрационные цели и может быть расширен для интеграции с реальным бэкендом и авторизацией через VK/Telegram.
